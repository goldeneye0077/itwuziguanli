# IT物资全生命周期管理系统｜前端路由与页面清单（草案）

默认采用 React 18 + TypeScript + Vite；UI 组件库 Ant Design；复杂表单优先 Drawer（说明书 UX 指南）。

## 1. 路由总览（建议）

### 1.1 公共

- `/login`：登录页
- `/`：根路由，重定向到 `/dashboard`

### 1.2 用户侧（User）

- `/dashboard`：智慧门户（Hero、公告、我的资产、快捷入口）
- `/store`：物资超市（分类树 + SKU 卡片）
- `/store/cart`：购物车页（可选；若采用悬浮购物车+抽屉结算可不单独出页）
- `/applications`：我的申请列表
- `/applications/:id`：申请详情（流转历史、明细、分配资产、提货券入口）
- `/assets`：我的资产列表（卡片/表格）
- `/assets/:id`：资产详情（SN、状态、历史流水）
- `/assets/return`：资产归还申请
- `/assets/repair`：故障报修申请
- `/pickup-ticket/:applicationId`：提货券（二维码 + 数字码）

### 1.3 审批与作业侧（Leader/Admin）

- `/approvals/leader`：领导初审待办
- `/approvals/admin`：管理员复审待办
- `/outbound`：出库作业台（Tab：待自取/待发货）
- `/inbound`：入库与库存管理（OCR 入库、SKU/Asset 管理、标签打印入口）
- `/assets/transfer`：资产转移（管理员/领导）
- `/admin/assets/scrap`：资产报废管理

### 1.4 分析与后台（Admin/Super Admin）

- `/analytics`：报表与分析（趋势、饼图、分布）
- `/copilot`：自然语言查询（提示词输入 + 结果表格/图表）
- `/admin/rbac`：RBAC 管理（角色、菜单、按钮权限）
- `/admin/crud`：数据库管理（可视化 CRUD 工具）

## 2. 页面组件清单（与说明书对齐）

### 2.1 智慧门户

- HeroBanner
- AnnouncementBoard
- MyAssetsCardList
- QuickActions（我要领用/故障报修/资产归还）

### 2.2 物资申领

- CategoryTree
- SkuCardGrid（展示图片、可用库存）
- FloatingCart（悬浮购物车）
- CheckoutDrawer（抽屉式结算：自取/快递切换 + 地址表单）
- AiPrecheckPanel（显示 AI 建议）

### 2.3 审批流

- ApprovalInbox（Leader/Admin 分页列表）
- ApplicationDetail（含明细与历史）
- LeaderReviewPanel（理由 + AI 建议 + 操作）
- AdminReviewPanel（分配 Asset、确认）

### 2.4 通知与验证/出库

- PickupQueueTab（扫码/输入码）
- PickupVerifyModalOrDrawer（明细、确认交付）
- ExpressQueueTab（发货录入）
- ShippingFormDrawer（快递单号、物流公司）
- PickupTicket（电影票样式：二维码 + 数字码）

### 2.5 入库与库存

- OcrUpload
- OcrReviewFormDrawer（识别结果 → 人工校对 → 入库）
- SkuCrudTable / AssetCrudTable
- AssetTagPrint（打印预览/下载）

### 2.6 报表与 Copilot

- AnalyticsDashboard（趋势图、饼图、分布图）
- CopilotQuery（问答输入 + 结果表格/图表）

### 2.7 资产生命周期

- ReturnAssetForm（归还申请）
- RepairRequestForm（故障报修）
- TransferAssetForm（资产转移）
- ScrapApprovalList（资产报废管理）
- AssetLifecycleTimeline（生命周期时间轴）
